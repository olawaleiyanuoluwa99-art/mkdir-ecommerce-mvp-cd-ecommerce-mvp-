# Modern E-Commerce Platform

A scalable, production-ready e-commerce platform built with modern web technologies, focusing on performance, security, maintainability, and an exceptional user experience.

## 🚀 Tech Stack

- Next.js 14+ (App Router)
- TypeScript (Strict Mode)
- React Query
- Zustand
- NestJS / Next.js Route Handlers
- PostgreSQL
- Prisma ORM
- Auth.js (NextAuth)
- Stripe Payments & Webhooks
- Sanity CMS
